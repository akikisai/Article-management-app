# Article-management-app

## 概要
ユーザーの記事を管理し、作成した記事に関してみんなが見れるを想定して作りました。（※練習用です）

Qiita記事にて、コード説明など振り返りという意味も込めて書きました。

[★記事はこちら](https://qiita.com/akikisai/items/ebc33cfa84ae2fbe0bfd)

## 機能
- アカウント登録＆ログイン
- 記事追加
- 記事編集
- 記事削除
- リーチエディタ

## 使用言語など
- Javascript
- node.js
- express
- MongoDB
- Git
- TinyMCE
